# Security Policy

For all projects under **RoboSoft Innovations (SMC-Private) Limited**.

## 📢 Reporting a Vulnerability
Email [robosoftinnovations@protonmail.com](mailto:robosoftinnovations@protonmail.com) with:
- Repository name
- Description and reproduction steps
- Impact assessment

Please do **not** open public issues for vulnerabilities.

## 🧠 Response Process
1. Acknowledge report within 48 hours  
2. Validate and prioritize  
3. Fix and patch  
4. Coordinate disclosure if required
